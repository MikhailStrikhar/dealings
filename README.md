# Dealings

Конфигурация 1С для учёта инвестиционных операций, позиций по инструментам, рыночных данных и финансового результата.

## Назначение

Проект предназначен для:

- учёта биржевых операций по инструментам
- хранения текущих позиций по счетам
- расчёта финансового результата по сделкам
- загрузки рыночных данных через внешние API
- накопления истории цен и свечей

## Основные возможности

- учёт операций покупки и продажи
- ввод начальных остатков
- хранение справочника инструментов и счетов
- загрузка операций из T-Invest API
- загрузка последних цен
- загрузка исторических свечей
- подготовка данных для анализа портфеля

## Архитектура

Проект разделён на несколько логических областей:

- учёт биржевых операций
- бизнес-логика расчёта позиций и финансового результата
- интеграция с внешними API
- загрузка и хранение рыночных данных

Подробная карта модулей и ответственности описана в `PROJECT_MAP.md`.

## Интеграции

Сейчас используется:

- T-Invest API / Tinkoff Invest API

В дальнейшем возможна интеграция с другими брокерскими платформами.
